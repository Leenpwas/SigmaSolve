# SigmaSolve — Advanced Engineering Calculator & Solver

SigmaSolve is a comprehensive Android application designed for engineering students and professionals. It combines a powerful scientific calculator with a guided problem-solving engine and a deep library of engineering formulas.

## 🚀 Features

- **Guided Problem Solvers**: Step-by-step interactive solvers for over 20 categories of engineering problems.
- **Formula Library**: A vast, searchable database of formulas organized by Year, Branch, and Subject.
- **Smart YouTube Integration**: Automatically searches for top educational content from **Neso Academy** and **Gate Smashers** for any topic.
- **Advanced Calculator**:
    - Scientific operations.
    - Real-time function graphing.
    - Matrix operations (Transpose, Determinant, Inverse, Rank).
    - Complex number support.
- **Study Tools**:
    - Bookmark formulas for quick reference.
    - "Recently Studied" history tracking.
- **Modern UI**: Built with Material Design 3 for a clean, intuitive experience.

## 📚 Engineering Topics Covered

1.  **Semiconductor Physics**: Intrinsic Concentration, Drift/Diffusion, Einstein Relation.
2.  **Diodes**: Current Equation, Thermal Voltage, Junction Capacitance.
3.  **Rectifiers**: Ripple Factor, Efficiency, PIV.
4.  **BJTs**: Alpha, Beta, Emitter Current, Transconductance.
5.  **MOSFETs**: Saturation Current, Transconductance.
6.  **Amplifiers**: Voltage Gain, Gain in dB.
7.  **Op-Amps**: Inverting/Non-Inverting Gain, Integrator, Differentiator.
8.  **Oscillators**: Barkhausen Criterion, Wien Bridge.
9.  **Filters**: Cutoff Frequency, Quality Factor.
10. **Digital Electronics**: De Morgan's, Consensus Theorem, Flip-Flops (SR/JK/D/T), Counters, Number Systems.
11. **Communication Systems**: Nyquist Rate, Shannon Capacity, Modulation Index.
12. **Signals & Transforms**: Fourier and Laplace Transforms.
13. **Transmission Lines**: Characteristic Impedance, Reflection Coefficient.
14. **ADC/DAC**: Step Size, DAC Output, SNR, Flash Comparators.
15. **Network Theory**: Ohm's Law, KCL, KVL, Thevenin, Norton, Max Power Transfer.
16. **Control Systems**: Closed Loop Gain, Damping Ratio, Natural Frequency.
17. **Microprocessors**: Instruction Cycle Timing (8085/8086).
18. **Antennas**: Gain, Effective Aperture.
19. **DSP**: Discrete Fourier Transform (DFT), Z-Transform.
20. **Computer Networks**: Sliding Window Efficiency.

## 🛠️ Tech Stack

- **Language**: Java
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Persistence Library
- **Navigation**: Jetpack Navigation Component
- **UI**: View Binding, Material Design 3, ConstraintLayout
- **Data**: Gson for initial database seeding

## 🏁 Getting Started

### Prerequisites
- Android Studio Iguana or newer.
- Android SDK 34 (API Level 34).
- Minimum API Level 28 (Android 9.0).

### Building the Project
1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Project with Gradle Files.
4. Click **Run** or use the terminal:
   ```bash
   ./gradlew installDebug
   ```
